🚀 DÉMARRAGE RAPIDE (3 MINUTES):

1. Installer dépendances:
   pip install paho-mqtt==2.1.0 flask flask-socketio eventlet psutil pyserial pillow

2. Lancer 4 services (4 terminaux):
   Terminal 1: mosquitto
   Terminal 2: python app_final_complete.py
   Terminal 3: python backbone_ultimate.py
   Terminal 4: python bridge_v2.py

3. Ouvrir navigateur:
   http://localhost:5000
