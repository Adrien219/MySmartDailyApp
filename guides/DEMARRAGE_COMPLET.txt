──────────────

  sudo systemctl start mosquitto
  sudo systemctl enable mosquitto

Vérifier:
  mosquitto --version

🎥 ÉTAPE 3: CONFIGURER ESP32-CAM (OPTIONNEL - 15 MIN)
═════════════════════════════════════════════════════════════════════════

Si vous avez un ESP32-CAM:

  1. Installer Arduino IDE
     https://www.arduino.cc/en/software

  2. Ajouter le support ESP32
     Fichier → Préférences → URL de gestionnaire de cartes:
     https://espressif-dl.cn/arduino_esp32/package_esp32_index.json

  3. Ouvrir le code
     Fichier → Ouvrir → esp32_firmware/esp32_camera_complete.ino

  4. Modifier les paramètres WiFi
     Ligne 8-10:
     const char* ssid = "VotreWiFi";           // ← Votre WiFi
     const char* password = "VotreMotDePasse"; // ← Votre mot de passe
     const char* mqtt_broker = "192.168.1.X";  // ← IP de votre PC/RPI

  5. Configurer Arduino
     Board: ESP32-CAM
     Upload Speed: 921600
     COM Port: Votre port (ex: COM3)

  6. Upload vers ESP32
     Sketch → Upload (Ctrl+U)

  7. Vérifier dans Serial Monitor (Outils → Serial Monitor)
     Doit afficher:
     ✅ Camera initialisée
     ✅ WiFi connecté!
     📍 IP: 192.168.1.Y

🆘 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════

❌ "Python not found"
   → Installer Python depuis python.org
   → Redémarrer PowerShell
   → Taper: python --version

❌ "pip not found"
   → Installer avec: py -m pip install --upgrade pip
   → Ou réinstaller Python en cochant "Add Python to PATH"

❌ "Mosquitto not found"
   → Télécharger depuis mosquitto.org
   → Installer la version Windows
   → Redémarrer PowerShell

❌ "MQTT connection refused"
   → Lancer Mosquitto EN PREMIER: mosquitto
   → Attendre: "mosquitto is running"
   → Relancer les autres services

❌ "Arduino not found"
   → Vérifier le port COM dans Gestionnaire des périphériques
   → Modifier le port dans bridge_v2.py si nécessaire
   → Brancher le câble USB
   → Redémarrer Arduino

❌ "Page blanche http://localhost:5000"
   → Vérifier Terminal 4: Flask doit afficher "Running on 0.0.0.0:5000"
   → Redémarrer Flask
   → Vérifier les erreurs dans le terminal

❌ "Profils ne se sauvegardent pas"
   → Vérifier que le dossier data/ existe
   → Vérifier les permissions du dossier
   → Vérifier que data/profiles.json est accessible

❌ "Téléphone ne peut pas accéder"
   → Vérifier sur le même WiFi
   → Utiliser l'IP correcte (ipconfig sur PC)
   → Vérifier que http://<IP>:5000 s'ouvre sur PC aussi

❌ "ESP32 ne se connecte pas"
   → Vérifier le WiFi de l'ESP32
   → Vérifier l'IP MQTT_broker dans le code Arduino
   → Vérifier que Mosquitto fonctionne
   → Redémarrer l'ESP32
